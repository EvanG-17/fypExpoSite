Web Based Deepfake Detection Application

Using Convolutional Neural Networks to detect manipulated media and prevent misinformation

What is this project?

This web-based deepfake detection application utilises AI neural networks to determine whether a piece of media is deep faked or not. The user uploads their file via the web application, the application returns a confidence percentage score, along with a definitive "True" or "False". This application helps users assess the authenticity of their media. Users can store their previous detections results for access at any time. This stored history can be valuable for various use cases, including media verification, forensic investigations, and digital content tracking.

How It Works

Users are able to upload their file, and instantly receive a confidence score.
